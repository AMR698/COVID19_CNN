# 🦠 تصنيف صور الأشعة السينية لمرض كوفيد-19 (COVID-19 CNN Classification)

هذا المشروع يهدف إلى بناء وتدريب نماذج تعلم عميق (Deep Learning) لتصنيف صور الأشعة السينية للصدر (Chest X-Ray) إلى ثلاث فئات: **COVID-19**، **Normal**، و **Viral Pneumonia**.

## 📋 وصف المشروع
يستخدم المشروع شبكات عصبية تلافيفية (CNN) لتحليل الصور الطبية والمساعدة في التشخيص التلقائي. يتضمن العمل مقارنة بين عدة بنى برمجية (Architectures) مشهورة في مجال الرؤية الحاسوبية:
- **CNN مخصص (Custom CNN)**: نموذج مبني من الصفر.
- **VGG16**: استخدام التعلم بنقل الخبرة (Transfer Learning) مع نموذج VGG المطور في جامعة أكسفورد.
- **ResNet101**: تجربة نموذج ResNet العميق من مايكروسوفت.

## 🛠️ التقنيات المستخدمة
- **اللغة**: Python
- **الإطار البرمجي**: TensorFlow / Keras
- **المكتبات الأساسية**:
  - `OpenCV` لمعالجة الصور.
  - `Pandas` و `NumPy` للتعامل مع البيانات.
  - `Matplotlib` و `Seaborn` للتصور البياني والنتائج.
  - `Scikit-learn` لتقييم الأداء وتقسيم البيانات.

## 📂 هيكل البيانات
يعتمد المشروع على مجموعة بيانات (Dataset) يتم تحميلها عبر مكتبة `opendatasets` من Kaggle، وتحتوي على مجلدات فرعية لكل فئة من فئات التصنيف.

## 🚀 كيفية التشغيل
1. قم بتثبيت المكتبات اللازمة باستخدام ملف `requirements.txt`:
   ```bash
   pip install -r requirements.txt
   ```
2. افتح ملف النوت بوك `COVID-19_CNN.ipynb` باستخدام Jupyter Notebook أو Google Colab.
3. اتبع الخطوات داخل النوت بوك لتحميل البيانات، تدريب النماذج، واستعراض النتائج.

## 📊 النتائج
يتضمن المشروع رسوماً بيانية توضح:
- دقة التدريب والتحقق (Training & Validation Accuracy).
- خسارة التدريب والتحقق (Training & Validation Loss).
- مصفوفة الارتباك (Confusion Matrix) لكل نموذج.
- تقرير التصنيف (Classification Report) الذي يوضح Precision و Recall و F1-score.
